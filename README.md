#  Custom Cursor Effect

A simple and interactive frontend project that replaces the default mouse cursor with a custom animated cursor that follows mouse movement in real-time.

---

##  Preview
A fullscreen black layout with a white circular cursor that smoothly follows the mouse pointer and blends with the background using CSS effects.

##  Features

---

##  Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (JS)**
---

##  How It Works
- JavaScript listens for `mousemove` events
- Cursor position updates dynamically using `clientX` and `clientY`
- Default system cursor is hidden
- Custom cursor is styled using CSS
